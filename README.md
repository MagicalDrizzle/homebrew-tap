# MagicalDrizzle Tap

## How do I install these formulae?

`brew install magicaldrizzle/tap/<formula>`

Or `brew tap magicaldrizzle/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "magicaldrizzle/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
